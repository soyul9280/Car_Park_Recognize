# 주차장 빈자리 인식 웹 
주차 자리를 분석해서 빈 자리를 확인해보세요 ! 🚗
## 소개
- 작업기간 | 2023. 09. 01 ~ 2023. 12. 20
- 구성원 | 3명
## 역할
- 프론트엔드 : goorm, bootstrap으로 웹 디자인
## 기술
- 차량 인식 : YOLO, OpenCV, Ultralytics
## 주요 기능
- 차량 인식 후 차량 인식 칸은 빨간색, 빈 자리는 초록색으로 표시
